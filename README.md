## How to Open in VS Code

1. Unzip the file `rotech_website.zip`
2. Open VS Code
3. File → Open Folder → select the `rotech` folder
4. Install the **Live Server** extension (by Ritwick Dey)
5. Right-click `index.html` → **Open with Live Server**
6. Website opens at `http://127.0.0.1:5500`

## How to Deploy (Free)

### Option 1 — Netlify (Recommended)
1. Go to https://netlify.com → Sign up free
2. Drag and drop the entire `rotech` folder onto Netlify
3. Your site is live instantly with a free URL
4. To use custom domain (e.g. rotechservice.in): Site Settings → Domain Management

### Option 2 — GitHub Pages
1. Create a free GitHub account
2. New repository → upload all files
3. Settings → Pages → Deploy from branch (main)
4. Site live at: https://yourusername.github.io/rotech
